Events are a collection of Races.

When creating an event, you should need all of the following:

- About you:
  - Name
  - Email
  - Phone number
- About the event:
  - Name
  - Description
  - Start date
  - End date
  - Location (address and coords stored in db)
  - Logo (image upload)
  - Website?
- Races[]: (at least one)
  - Name
  - Description
  - Length (type 5K, 10K, etc...)
  - Race start time
